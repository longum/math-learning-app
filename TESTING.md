## 🔍 故障排查

### 如果页面显示空白

**检查浏览器控制台**:
1. 打开浏览器
2. 按 `F12` 或右键点击 → "检查"
3. 查看 Console 标签是否有错误

**常见错误**:
- JavaScript 错误（红字）
- 404 错误（找不到文件）
- 网络错误

### 如果无法访问服务器

**重启服务器**:
```bash
pkill -f "vite.*dev"
npm run dev
```

**查看日志**:
```bash
lsof -ti:5173  # 检查端口是否被占用
```
